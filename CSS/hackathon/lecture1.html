Lecture 1 – HTML Basics

Introduction

HTML stands for HyperText Markup Language. It structures content on the web using tags.

Headings & Paragraphs

Use <h1> to <h6> for headings and <p> for paragraphs.

Links & Images

<a href="url">Link</a> creates hyperlinks.

<img src="image.jpg" alt="Description"> adds images.
