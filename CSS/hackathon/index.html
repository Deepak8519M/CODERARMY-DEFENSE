Welcome to My Web Development Notes

This hackathon project is built using only HTML, featuring multiple lectures, quizzes, embedded videos/audio, and interactive notes. Enjoy exploring!

Features

  • Documentation-style sidebar navigation
  • Collapsible lecture notes using <details> & <summary>
  • Embedded videos and audio
  • Quizzes for self-revision
  • Timeline of learning progress
  • Google Fonts integration
  • Tables used for layout and highlights
